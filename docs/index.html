FAQ

Common questions about Deskrona

Is my data sent to any server?
No. Deskrona is local-first. All tracking data, screenshots, and settings are stored in a SQLite database on your machine. Nothing leaves your computer.
Which platforms are supported?
Currently Windows (64-bit). Cross-platform support for macOS and Linux is planned for future releases.
Does it track browser URLs?
Yes, Deskrona can detect browser URLs from Chrome, Firefox, Edge, Brave, and other Chromium-based browsers. This can be configured in settings.
Can I use it with my team?
Yes. Deskrona supports multi-user mode with company admin and employee accounts. Admins can view team productivity and manage settings centrally.
How often are screenshots taken?
The screenshot interval is configurable in settings (default: 10 seconds). You can also disable screenshots entirely if preferred.
Is there a backup feature?
Yes. Automated backups can be scheduled (daily, weekly, monthly) with a configurable backup location. Manual export and import are also supported.
